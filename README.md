Задача №5

Запустить программу можно через компилятор на гитхабе, собирать ничего не надо, всё одним файлом.
Вид ввода : вводите в порядке кучек (с самой нижгней карты первой кучки по верхнюю карту 8-ой кучки), номинал карты ("туз", "король", "дама", "валет", "десятка", "девятка", "восьмёрка", "семёрка", "шестёрка") с маленькой буквы в иминительном падеже, далее вводите масть ("пик", "бубен", , "черв", "крести"). Остальное делает программа.

Задача решалась через поиск в ширину, каждой ситуации присваивали уникальный номер, чтобы если будем возвращаться в эту вершину мы могли бы это идентифицировать(что посещенная вершина). Поиск в ширину работает за O(V + E), где первая буква обозначает - количество вершин, вторая ребра.
